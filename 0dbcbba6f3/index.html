軸の補足説明:
状態 = 追い切り評価が中心。1週前・最終追切の動き／時計、馬体重推移、当日パドック気配、陣営コメントを含む総合的なコンディション。
血統 = 父・母父の距離・馬場・コース適性。実績の薄い馬（未勝利・新馬・初コース）ほど重み大。
ローテ = 前走からの間隔・休養明け・連戦疲労の評価。
前走補正 = 前走の結果（特に敗戦）をどれだけ「事故」として割引いて評価すべきか。不利・進路皆無・落鉄・極端ペース被害など。「度外視できる材料があるほど高得点」。

計算ロジック:

加重合計 = Σ(採点 × weight) / Σ(採点した軸の weight) ← 加重平均
  weight (G1): 近走0.20 / クラス0.15 / コース距離0.15 / 状態0.13 / 騎手0.08 / 馬場0.08 / 枠展開0.06 / 血統0.08 / ローテ0.05 / 度外視0.02
  weight (平場): 血統 0.10、状態 0.10 等で再配分
  weight (未勝利・新馬): 血統 0.25・状態 0.18・騎手 0.13 で再配分
↓ 補正（adjustments）を加算（補正履歴・evidence_ids 必須 = 隠れ補正禁止）
補正後 = 加重合計 + Σ(補正の差分)
情報量 = 軸ごとに集められた情報量（追切評価が複数あるか、過去走の本数、騎手データの蓄積など）の平均
  0/5＝そもそも採点できる材料がない 〜 5/5＝一次情報まで揃って確信を持って採点できる
↓
総合スコア = 補正後 × (情報量 / 5) ← 印選定に使う最終値
  同じ採点でも、材料が乏しい馬は控えめ評価になる仕組み（過信防止）

今回の学び

馬場想定リスク台風・大雨で道悪想定でも当日回復で良馬場になるケースがあり、道悪前提の予想は当日馬場更新で必ず再点検すべき。
人気薄先行の警戒外差し馬場でも道中でペースが緩むと前残りになる。8人気シックスペンスの好位抜け出しを軽視した(△)のが敗因で、先行力ある人気薄は評価を上げるべき。
大外枠のマイルG1○トロヴァトーレは17番大外枠で出遅れと外々のロスで9着。東京マイルG1の大外枠は割引材料として重く見るべき。
