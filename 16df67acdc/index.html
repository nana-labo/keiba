ひと言で: 印（◎○▲△☆）を直感や文章判断ではなく、7 つの観点ごとに 0〜5 点で採点し、点数の順位から印を機械的に決める仕組み。同じ情報を入れれば誰がやっても同じ点数になる＝再現性が高い。

軸の補足説明:
状態 = 追い切り評価が中心。1週前・最終追切の動き／時計、馬体重推移、当日パドック気配、陣営コメントを含む総合的なコンディション。
血統 = 父・母父の距離・馬場・コース適性。実績の薄い馬（未勝利・新馬・初コース）ほど重み大。
ローテ = 前走からの間隔・休養明け・連戦疲労の評価。
前走補正 = 前走の結果（特に敗戦）をどれだけ「事故」として割引いて評価すべきか。不利・進路皆無・落鉄・極端ペース被害など。「度外視できる材料があるほど高得点」。

計算ロジック:

加重合計 = Σ(採点 × weight) / Σ(採点した軸の weight) ← 加重平均
  weight (G1): 近走0.20 / クラス0.15 / コース距離0.15 / 状態0.13 / 騎手0.08 / 馬場0.08 / 枠展開0.06 / 血統0.08 / ローテ0.05 / 度外視0.02
  weight (平場): 血統 0.10、状態 0.10 等で再配分
  weight (未勝利・新馬): 血統 0.25・状態 0.18・騎手 0.13 で再配分
↓ 補正（adjustments）を加算（補正履歴・evidence_ids 必須 = 隠れ補正禁止）
補正後 = 加重合計 + Σ(補正の差分)
情報量 = 軸ごとに集められた情報量（追切評価が複数あるか、過去走の本数、騎手データの蓄積など）の平均
  0/5＝そもそも採点できる材料がない 〜 5/5＝一次情報まで揃って確信を持って採点できる
↓
総合スコア = 補正後 × (情報量 / 5) ← 印選定に使う最終値
  同じ採点でも、材料が乏しい馬は控えめ評価になる仕組み（過信防止）

事実情報（枠・騎手・血統・近走）は一次情報に基づきます。能力評価とシミュレーション結果は分析からの推定で、的中を保証するものではありません。馬券は20歳以上が自己責任の範囲で。

予想 vs 結果

馬場予想: 良想定
結果: レース直前の大雨で良から重へ急変。パワー馬場となり重巧者・逃げ先行が浮上。
ペース予想: 標準的なミドル想定
結果: 実際もM。コスモキュランダ先行で前々有利、上がり最速35.0でも差し届かず前残りの決着。
印判定予想: ◎クロワデュノール本命
結果: ◎クロワデュノールは1番人気で2着(クビ差)と好走も勝ち切れず。勝ったのは△評価のメイショウタバル(16番)。○ミュージアムマイルは9着と振るわず、▲ダノンデサイルは3着で上位来。重馬場適性の見誤りが最大の敗因。

◎ クロワデュノール2着 / 不発

1番人気に応え好位5番手から直線で猛追もクビ差届かず。春古馬三冠ならず。能力は示したが重馬場と前有利展開が向かなかった。

○ ミュージアムマイル9着 / 不発

4番人気・レーン騎乗も上がり36.5と伸びを欠き二桁着順に沈む。重馬場が明確に不向き。

▲ ダノンデサイル3着 / 的中

3番人気で堅実に3着確保。上がり最速35.0の脚を使うも前を捕えきれず。複勝圏内は死守。

△ メイショウタバル1着 / 的中

最大の抑え評価が的中。父ゴールドシップ譲りの重馬場巧者ぶりを発揮し2番手抜け出しで連覇。恵みの雨が味方した。

△ レガレイラ7着 / 不発

5番人気・ルメール騎乗も牝馬に重馬場は堪え掲示板外。見せ場薄く7着。

今回の学び

馬場急変リスク直前の大雨で良から重へ急変。父ゴールドシップ産駒(メイショウタバル)等の重馬場巧者が一気に浮上。天候予報と血統の重適性を印に強く反映すべき。
本命の取捨◎クロワデュノールは能力最上位でも重馬場×前有利展開で差し届かず2着。良馬場前提の絶対評価は馬場悪化で崩れる。馬場別の相対評価を持つこと。
事前データ照合事前予想の馬番(☆シンエンペラー11番等)が実結果の馬番割当と一部食い違い。出馬確定後に馬番を再照合する運用が必須。
